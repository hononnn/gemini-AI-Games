# 🏮 重生之我在古代卖年货 (Lunar Hustle 2.5)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Tech: React](https://img.shields.io/badge/Tech-React-61DAFB?logo=react)](https://reactjs.org/)

> **“腊月寒冬，碎雪迷眼。且看你如何凭手中八两碎银，博一份泼天富贵！”**

这是一款基于 React 开发的古风模拟经营 + 文字推演类小游戏。玩家将在 14 天的年关期限内，通过策略贸易、人脉经营与随机事件抉择，体验一场古代商贩的逆袭之路。

---

## 📖 游戏策划文档 (Game Design Document)

### 1. 核心概念 (Core Concept)
*   **背景：** 大庆历三十年，玩家扮演一名初入青石镇的小商贩。
*   **目标：** 在 14 天内，通过低买高卖年货积累财富，同时提升名望与狡诈度，达成不同的社会等级结局。
*   **核心挑战：** 每日固定 20 两的生存成本，以及不可控的体力（气力）分配。

### 2. 核心系统逻辑 (Core Systems)

#### 🎲 气力系统 (AP System)
*   **骰子驱动：** 每日开始时通过掷骰决定当日体力值（1-6 点）。
*   **消耗规则：** 
    *   **进货：** 首次购买消耗 1 点，连续购买同类商品 **不消耗**。
    *   **出货/休息/社交：** 每次操作固定消耗 1 点。
*   **设计目的：** 增加 Roguelite 的随机性，玩家必须在有限的步数内优化利润。

#### 📈 经济平衡 (Economic Balance)
*   **商品体系：** 从廉价的“红纸爆竹”到昂贵的“镇宅瑞兽”，共 5 级阶梯。
*   **动态物价：** 
    *   **流言板：** 每日刷新市场流言，特定商品价格产生 0.5x - 2.0x 波动。
    *   **环境加成：** 大雪天气（Snowy）提升高价值商品的卖出利润。
*   **生存压力：** 每日强制结算 20 两伙食费。银两不足会导致饥饿，连续两日饥饿则游戏提前结束。

#### 🎭 社交与推演 (Social & RPG)
*   **NPC 交互：** 三大阵营 NPC（权势、竞争对手、地主）。
*   **走关系：** 进入社交界面需消耗气力。身无分文会触发“被撵走”事件并降低好感。
*   **文字推演：** 交易过程中随机触发由 Gemini API (或本地库) 驱动的剧情事件，选项影响多维属性。

### 3. 技术栈 (Tech Stack)
*   **前端框架：** React 19 (Hooks 驱动状态管理)
*   **样式处理：** Tailwind CSS (极简主义与中国传统色)
*   **交互支持：** 
    *   **Gamepad API：** 支持标准游戏手柄操作。
    *   **Gemini API：** 驱动智能化、动态生成的事件描述与对话内容。

### 4. 视觉与交互设计 (UI/UX)
*   **传统色彩：** 采用“杏仁色”纸张背景、“朱砂红”强调色与“松烟墨”文字。
*   **响应式布局：** 严格锁定 16:9 比例，支持全屏剧场模式。
*   **操作反馈：** 集市网格焦点自动滚动（Scroll-into-view），支持手柄连击操作。

---

## 🎮 操作指南
*   **键盘：** `方向键` 移动焦点，`Enter` 确认操作，`Esc` 退出/后退。
*   **手柄：** `十字键/摇杆` 移动，`A键` 确认，`B键` 退出。
*   **特别技巧：** 关注屏幕上方的“流言板”，这是低买高卖的关键！
