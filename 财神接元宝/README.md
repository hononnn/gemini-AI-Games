# 财神接元宝

一款以中国传统财神接元宝为主题的休闲小游戏，结合了趣味性和传统文化元素。

## 游戏策划思路

### 核心概念
- **传统文化**：以中国传统财神形象和元宝为核心，弘扬传统文化
- **休闲玩法**：设计简单易上手的游戏机制，适合各年龄段玩家
- **节日氛围**：营造浓厚的节日气氛，特别适合在春节等传统节日游玩

### 设计目标
- 创造一款轻松愉快的休闲游戏
- 传承和弘扬中国传统文化元素
- 提供简单直观的游戏体验和流畅的操作感

## 游戏玩法

### 游戏机制
1. **游戏开始**：玩家进入游戏后，系统会初始化游戏状态
2. **财神移动**：玩家通过操作控制财神左右移动
3. **元宝掉落**：游戏会不断从上方掉落元宝
4. **接元宝**：玩家需要控制财神接住掉落的元宝
5. **计分系统**：根据接住的元宝数量计算分数
6. **游戏目标**：在规定时间内接住尽可能多的元宝，获得高分

### 操作方式
- 使用键盘左右方向键控制财神移动
- 使用鼠标移动控制财神位置
- 遵循游戏界面上的提示进行操作

## 技术实现

### 技术栈
- **前端技术**：HTML5 + CSS3 + JavaScript
- **图像资源**：传统财神形象和元宝元素
- **动画效果**：CSS动画和JavaScript交互

### 核心功能
- `game.html`：游戏主页面结构
- `game.css`：游戏样式和视觉效果
- `game.js`：游戏逻辑和交互功能

## 运行说明

### 前置条件
- 现代Web浏览器（Chrome、Firefox、Safari等）

### 运行方式
1. 直接在浏览器中打开`game.html`文件
2. 开始游戏并享受接元宝的乐趣
